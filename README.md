# Firebase Studio

This is a NextJS starter in Firebase Studio.

To get started, take a look at src/app/page.tsx.

## Color Palette

The application primarily uses a dark theme. Here are the key colors used throughout the UI:

| Role        | English Name       | Hex Code  | HSL Value          | Used For                                      |
|-------------|--------------------|-----------|--------------------|-----------------------------------------------|
| Background  | Midnight Blue      | `#10172A` | `222 47% 11%`      | Main app background.                          |
| Foreground  | Almost White       | `#FAFAFA` | `0 0% 98%`         | Main text color.                              |
| Primary     | Vivid Violet       | `#8A4DFF` | `262 82% 57%`      | Logos, key buttons, icons, and highlights.    |
| Secondary   | Dark Gunmetal      | `#1D283A` | `217 33% 17%`      | Less prominent UI elements and card backgrounds. |
| Accent      | Bright Teal        | `#00C1C1` | `190 100% 38%`     | Secondary actions, like saving drafts.        |
| Border      | Slate Gray         | `#26364D` | `217 33% 22%`      | Borders and input field backgrounds.          |
| Destructive | Dark Red           | `#8C1F1F` | `0 63% 31%`        | Deletion and logout actions.                  |
