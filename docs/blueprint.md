# **App Name**: Social Sage

## Core Features:

- Topic Research: Research a topic provided by the user using an AI tool and summarize key information.
- Twitter Post Generation: Generate multiple Twitter posts based on the researched topic, with varying tones and angles, using an AI tool.
- LinkedIn Post Generation: Generate multiple LinkedIn posts based on the researched topic, with a professional tone, using an AI tool.
- Post Selection and Editing: Allow users to view, edit, and select the generated posts for publishing.
- Copy to Clipboard: Provide a simple interface to copy the selected posts to the user's clipboard.

## Style Guidelines:

- Primary color: Deep blue (#1A237E) for trustworthiness and professionalism.
- Secondary color: Light gray (#EEEEEE) for clean backgrounds.
- Accent: Teal (#00ACC1) for highlights and action buttons.
- Clean and modern layout with a focus on readability and ease of use.
- Use clear and concise icons to represent different features and actions.
- Subtle animations for loading states and transitions to improve user experience.
